# 🧪 QA 에이전트 (Quality Assurance Agent)

브라우저를 사용하여 QuizRank.io의 주요 페이지들을 순회하며, UI 깨짐·기능 오류·링크 에러 등을 자동으로 점검합니다.

## 실행 방법

사용자가 `/qa` 또는 `/qa [특정 페이지]` 형태로 입력합니다.
예시:
- `/qa` — 전체 페이지 점검
- `/qa worldcup` — 월드컵 관련 페이지만 점검

## 워크플로우 단계

### 1단계: 로컬 서버 구동

로컬 개발 서버를 실행합니다. 이미 실행 중이면 이 단계를 건너뜁니다.

```sh
cd C:\Users\main\Desktop\Dev\products\quiz-platform
npx wrangler dev
```

### 2단계: 페이지 순회 점검

아래 체크리스트에 따라 각 페이지를 브라우저로 방문하고 검증합니다.

#### 📋 점검 체크리스트

| 페이지 | URL (로컬) | 확인 항목 |
|--------|-----------|-----------|
| 메인 홈 | `http://localhost:8787/` | 퀴즈·월드컵·티어 카드 렌더링, 이미지 로딩, 인기순 정렬 |
| 퀴즈 목록 | `/quiz-list.html` | 퀴즈 카드 목록 표시, 필터링 동작 |
| 퀴즈 플레이 | `/quiz-play.html?slug=[아무_슬러그]` | 문제 표시, 보기 클릭, 정답 판정, 점수 기록 |
| 월드컵 목록 | `/worldcup-list.html` | 월드컵 카드 렌더링, 이미지 로딩 |
| 월드컵 플레이 | `/worldcup-play.html?slug=[아무_슬러그]` | 라운드 진행, 이미지 표시, 선택 동작 |
| 티어 리스트 | `/tier-list.html` | 티어 목록 표시, 카드 렌더링 |
| 로그인 | `/login.html` | 폼 렌더링, 입력 필드 동작 |
| 관리자 | `/admin-dashboard.html` | 대시보드 로딩 (로그인 필요) |

#### 🔍 공통 점검 항목
- [ ] **이미지 엑박 없음**: 모든 이미지가 정상 로딩되는가
- [ ] **콘솔 에러 없음**: 브라우저 콘솔에 JavaScript 에러가 없는가
- [ ] **반응형 레이아웃**: 모바일 뷰(375px)에서 UI가 깨지지 않는가
- [ ] **다크 모드**: 다크 모드 테마가 올바르게 적용되는가
- [ ] **링크 동작**: 네비게이션 링크가 모두 정상 동작하는가
- [ ] **API 응답**: 데이터가 정상적으로 불러와지는가

### 3단계: 결과 보고

점검 결과를 아래 형식으로 정리하여 사용자에게 보고합니다:

```markdown
## 🧪 QA 결과 보고서

### ✅ 정상 (N개 페이지)
- 메인 홈: 정상
- 퀴즈 목록: 정상
...

### ❌ 이슈 발견 (N건)
1. **[페이지명]** — [이슈 설명] (심각도: 높음/중간/낮음)
2. ...

### 💡 개선 제안
- [발견된 개선 포인트]
```

## 핵심 규칙
1. **스크린샷 첨부**: 이슈 발견 시 해당 화면을 캡처하여 보고에 포함합니다.
2. **콘솔 로그 확인**: 각 페이지의 브라우저 콘솔을 반드시 확인합니다.
3. **모바일 뷰 필수**: 반응형 테스트는 375px 너비로 수행합니다.
4. **한국어로**: 모든 보고를 한국어로 작성합니다.
