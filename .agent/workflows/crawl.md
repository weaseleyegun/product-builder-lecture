# 🤖 데이터 수집(Crawl) 에이전트

주제(Prompt)와 예시 1개를 입력하면, AI가 자동으로 퀴즈 정답과 유튜브 검색어를 생성하고 대응되는 유튜브 영상을 수집/검증한 뒤 데이터베이스에 완성된 퀴즈셋을 배포합니다.

## 실행 방법

사용자가 `/crawl [주제] | [예시] | [수집개수]` 형태로 파이프(`|`)로 구분지어 입력합니다.
예시: `/crawl 2000년대 웹툰 | 마음의 소리 | 10`

만약 사용자가 단순히 `/crawl` 이라고만 입력했다면, 사용자에게 `주제`, `예시`, `개수`를 차례대로 물어보거나 하나로 묶어서 물어본 뒤 진행하세요.

## 워크플로우 단계

// turbo-all

### 1단계: 파라미터 파싱 및 검증

사용자 입력에서 파라미터 3가지를 추출합니다. (예: `주제: 2000년대 웹툰`, `예시: 마음의 소리`, `수집개수: 10`)

### 2단계: 크롤링 스크립트 실행

추출한 인자를 바탕으로 커맨드 라인에서 크롤링 봇을 실행합니다. `--yes` 플래그를 붙이면 봇이 첫 번째 AI 추천 제목을 자동 선택하고 터미널의 재차 확인 절차 없이 즉시 DB에 적재합니다.

```sh
cd C:\Users\main\Desktop\Dev\products\quiz-platform
node scripts/crawling_agent.js "[주제]" "[예시]" [수집개수] --yes
```

- 스크립트 실행 중 에러가 발생하면 멈추고 사용자에게 에러를 알립니다.

### 3단계: 결과 보고

스크립트가 정상적으로 종료되면 사용자에게 완료를 보고합니다:

```
✅ 지식 수집 및 퀴즈 자동 발행을 완료했습니다!
- 주제: [주제]
- 수집 요청: [수집개수]개
👉 생성된 퀴즈는 메인 화면에서 확인하실 수 있습니다.
```
