# 🧹 퀴즈 플랫폼 DB 일괄 클린업

데이터베이스의 퀴즈 문제들 중에서 오염된 보기 (`A`, `B`, `D`, `Ad` 등 의미 없는 영단어) 또는 정답과 보기에 보기 싫은 형식(예: `가수명 - 노래제목`)이 발견되었을 때 이를 무작위 안전한 데이터로 동적 교체하고 깔끔하게 텍스트를 파싱(Parsing)하여 복구하는 일괄 정제 자동화 스크립트입니다.

// turbo-all
1. `quiz-platform` 작업 디렉터리로 이동하여 Node.js 스크립트를 실행합니다.

```sh
cd C:\Users\main\Desktop\Dev\products\quiz-platform
node scripts/db_cleanup.js
```

### 💡 스크립트 상세 설명
- **가수명 파싱:** 모든 문제의 정답 및 보기를 스캔하여 `가수 - 노래제목` 포맷을 `노래제목`으로 일괄 통일합니다.
- **오염 데이터 적발:** `options`에 영단어 `A, B, C, Ad` 등의 스크립트 버그 흔적이 존재하면 이를 즉각 식별합니다.
- **안전한 재구성:** 버그가 발견된 문항의 보기를 전부 버리고 현재 DB 내의 가장 정상적인 다른 노래 제목 3개를 무작위로 추첨하여 **오답 보기(Wrongs)** 로 채워 넣은 뒤 정상 렌더링되게 복원시킵니다.
- **소요 시간:** 데이터 개수에 따라 약 15~30초 소요됩니다.
