# QuizRank.io 🏆 / (https://quizrank.pages.dev)

재치 있는 유튜브 구간 맞추기 퀴즈와 고퀄리티 이상형 월드컵(실시간 API 연동 및 유저 생성형 티어표)을 제공하는 인터랙티브 웹 플랫폼입니다.

## 🚀 프로젝트 목표 및 주요 기능

### 1. 🎵 퀴즈 (Quiz Play)
- 유튜브 영상의 **특정 구간(Start~End초)** 을 자동 반복 재생하여 시청한 후, 관련된 4지선다형 정답을 맞추는 기능입니다.
- **오픈소스 영상(NCS, 뮤직비디오 등)** 의 구간들을 직접 매핑하여 흥미로운 사용자 경험을 유도합니다.

### 2. 🔥 이상형 월드컵 (Worldcup Tournament)
- 글로벌 공공 무료 API(Jikan Anime, PokeAPI, DogAPI)와 실시간 연동하여 **가장 인기 있는 데이터**를 직접 수집해옵니다.
- 사용자는 최소 4강부터 최대 256강까지 **원하는 라운드 옵션을 설정**하여 우승자를 가리는 토너먼트 게임을 즐길 수 있습니다.

### 3. 🏅 나만의 티어 및 랭킹 만들기 (Creation & UGC)
- 유저들이 직접 폰이나 컴퓨터의 이미지 파일을 업로드하고, **S, A, B, C, D 급으로 드래그 앤 드롭** 하여 나만의 티어표를 완성합니다.

---

## 🛠 기술 스택 (Tech Stack)

* **프론트엔드 (Frontend):** HTML5, Vanilla JavaScript, CSS3 (CSS Variables 기반 Light/Dark 테마 지원)
* **백엔드 (API & DB):** Cloudflare Workers (서버리스), Supabase (PostgreSQL & Auth)
* **API 및 도구:** 유튜브 IFrame API (비디오 재생), Fetch API (오픈 데이터 수집)

## 💻 로컬에서 프로젝트 실행하기

별도의 무거운 자바스크립트 프레임워크(React, Vue 등) 없이 코드가 구성되어 있어, 간단한 정적 서버 도구만으로도 쾌적하게 실행할 수 있습니다.

1. **저장소 폴더**로 이동합니다.
2. 터미널을 열고 **의존성 설치 및 로컬 서버를 실행**합니다.
   ```bash
   # 로컬 웹 서버 실행 (NodeJS가 설치되어 있어야 합니다)
   npx serve .
   ```
3. 터미널에 나타나는 로컬 주소(예: `http://localhost:3000` 또는 터미널에 찍히는 번호)를 **웹 브라우저로 오픈**합니다.
